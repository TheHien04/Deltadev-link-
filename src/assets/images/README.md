# Image Guidelines for DeltaDev Link Website

This folder should contain the following images for the website:

## Required Images

### 1. hero-sausage.jpg
- **Location**: Hero section (top of homepage)
- **Recommended size**: 1200 x 1100 px
- **Description**: Beautiful shot of cooked lap xuong with rice/garnish
- **From your images**: Use the first image (sausages on banana leaf with rice)

### 2. sausage-packages.jpg
- **Location**: About section
- **Recommended size**: 1000 x 1000 px
- **Description**: Multiple packages of lap xuong with yellow labels
- **From your images**: Use the second image (packaged sausages with labels)

### 3. certificate.jpg
- **Location**: Quality/Certificate section
- **Recommended size**: 800 x 1000 px
- **Description**: Food safety certificate
- **From your images**: Use the third image (certificate document)

### 4. product-sausage.jpg
- **Location**: Product card section
- **Recommended size**: 800 x 600 px
- **Description**: Close-up of fresh sausages
- **From your images**: Use the fourth image (red basket of sausages)

## Image Optimization Tips

1. **Format**: Use JPG for photos (smaller file size)
2. **Compression**: Compress images to reduce loading time (use tools like TinyPNG)
3. **Quality**: Balance between quality and file size (aim for under 200KB per image)
4. **Naming**: Keep file names exactly as listed above

## How to Add Images

1. Save your images with the exact names listed above
2. Place them in this `images` folder
3. The website will automatically display them

## Alternative Image Sources

If you don't have these specific images ready, the website will show colored placeholders with gradient backgrounds until you add the actual photos.

---

**Note**: The website is designed to work with or without images. Add them when ready!
